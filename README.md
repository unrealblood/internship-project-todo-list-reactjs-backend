This is a [Node.js](https://nodejs.org/) and [Express.js](https://expressjs.com/) internship project (ToDo List) backend, currently in development and not hosted on any server.

## Getting Started

1. First run:
```bash
npm install
```
To install all packages and dependencies.

2. Then run:
```bash
npm run dev
```
To start the nodejs server.

3. Now, open [http://localhost:5173](http://localhost:5173) along with reactjs frontend which is here: [TodoList frontend](https://github.com/unrealblood/internship-project-todo-list-reactjs-frontend/) in your browser to see the project website.
